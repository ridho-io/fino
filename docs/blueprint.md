# **App Name**: Fino

## Core Features:

- Intelligent Finance Dashboard: A high-fidelity dashboard for React Native featuring a swipeable wallet carousel, mini cash-flow charts, and AI-driven insights using a premium glass-morphism aesthetic.
- Natural Language Bookkeeping: A conversational tool that allows users to record transactions via voice or text prompts, automatically parsing amount, category, and wallet details using generative models.
- Vision-Based Receipt Scanner: Uses an AI tool to extract data from receipt images, identifying merchants, dates, and line items to automate transaction entry within the Expo environment.
- Precision Transaction Engine: Comprehensive manual entry system supporting multi-currency, split transactions, and recurring bills with a sleek card-based interface optimized for mobile performance.
- Proactive AI Financial Agent: A dedicated chat tool for financial queries, anomaly detection, and proactive spending advice based on local transaction history.
- Smart Budgeting & Goals: Visual goal tracking and category-based budgeting with rollover logic, progress rings, and proximity alerts using Reanimated for smooth interactions.
- Local-First Secure Database: Privacy-first architecture using encrypted on-device expo-sqlite and Drizzle ORM for all financial data, with optional encrypted backup.
- OS-Native Integration: Deep integration with system features including biometric authentication via expo-local-authentication and native iOS-style context menus.

## Style Guidelines:

- Default palette is centered on Deep Forest Teal (#1A312C) and Muted Sage (#428475) to evoke a sense of premium financial stability.
- Accent highlights use Mint Green (#89D7B7) for positive growth trends and interactive elements, with support for customizable accent themes in settings.
- Surface and text clarity is maintained using Creamy Off-white (#FFF4E1) to contrast against the deep background tones.
- Utilizes 'SF Pro' system fonts for a native iOS feel, with 'Inter' as a high-legibility cross-platform fallback for numerical data.
- Custom precision line-style icons with rounded caps and consistent stroke weights to match the modern Expo UI aesthetic.
- Card-based layout with large corner radii (24px) and heavy use of 'expo-blur' for frosted glass effects on headers and tab bars.
- Fluid, spring-based transitions powered by react-native-reanimated for all navigation and interactive state changes.
